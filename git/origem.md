# 🌍 Origem do Git

## Quando e por que o Git foi criado?

O Git foi criado em **2005** por **Linus Torvalds**, o criador do kernel Linux, em meio a uma crise no processo de desenvolvimento do próprio Linux. [web:3][web:17]

Na época, o time usava o sistema de versionamento **BitKeeper**, que era proprietário. Quando o suporte gratuito foi retirado, ficou claro que era preciso uma alternativa própria, rápida e confiável para continuar a evolução do kernel.

### 🕰️ Contexto histórico

- **2005**: BitKeeper muda de licença e deixa de ser gratuito para o desenvolvimento do Linux.
- Em resposta, **Linus Torvalds** decide criar uma nova ferramenta do zero.
- O Git nasce em apenas **10 dias** de desenvolvimento intensivo. [web:3][web:17]

## 🎯 Motivação principal

O Git foi pensado para resolver três necessidades centrais no desenvolvimento do Linux:

- **Velocidade**: comutar (commitar) mudanças em milissegundos, mesmo em repositórios muito grandes.
- **Distribuído**: funcionar totalmente offline, sem depender de um servidor central.
- **Descentralizado**: permitir que vários desenvolvedores mantenham cópias completas do histórico, sem um único ponto de falha. [web:3][web:17]

> **"Eu queria algo que fosse realmente rápido."**  
> — Linus Torvalds

# 🐢 Git — Sistema de Controle de Versão

Esta pasta reúne os principais conteúdos sobre o **Git**, a ferramenta de controle de versão distribuído que revolucionou o desenvolvimento de software e o trabalho colaborativo entre desenvolvedores.

Aqui você encontrará explicações teóricas, exemplos práticos e os comandos fundamentais utilizados no dia a dia de projetos profissionais.

---

## 📋 Conteúdo desta pasta

| Arquivo | Descrição |
|----------|------------|
| [`origem.md`](./origem.md) | História, criação e evolução do Git |
| [`o_que_faz.md`](./o_que_faz.md) | Funcionamento do Git e conceitos fundamentais |
| [`comandos_basicos.md`](./comandos_basicos.md) | Principais comandos utilizados no terminal |

---

## 🚀 Objetivo

O objetivo desta seção é apresentar os fundamentos do Git de forma clara e prática, permitindo que qualquer pessoa consiga:

- Entender o funcionamento do versionamento
- Criar repositórios
- Registrar alterações
- Trabalhar com commits
- Organizar projetos de maneira profissional

---

## 💻 Recomendação

Leia os arquivos na ordem apresentada e pratique cada comando diretamente no terminal para fixar o aprendizado.

```bash
git init
git add .
git commit -m "Meu primeiro commit"
```

A prática é essencial para dominar o Git 🚀

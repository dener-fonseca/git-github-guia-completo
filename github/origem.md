# Origem do GitHub

## Quando e como o GitHub nasceu?

O GitHub foi lançado em **19 de outubro de 2008** por **Tom Preston‑Werner**, **Chris Wanstrath** e **PJ Hyett**, desenvolvedores que queriam facilitar a colaboração com Git. [web:3][web:6]

Por muitos anos, o Git era usado localmente, mas não havia uma plataforma simples e visual para:

- hospedar repositórios;
- permitir que outras pessoas contribuíssem;
- discutir código e mudanças.

O GitHub resolveu justamente isso: uma **interface web** em cima do Git, com funcionalidades sociais e colaborativas.

## Eventos importantes

- **2008**: lançamento do GitHub.
- **2018**: a Microsoft adquiriu o GitHub por aproximadamente 7,5 bilhões de dólares.
- **Presente**: GitHub é uma das principais plataformas de código aberto do mundo, usada por milhões de desenvolvedores. [web:3][web:6]

---

> Resumo: o GitHub nasceu para tornar o Git mais acessível e colaborativo, transformando repositórios locais em plataformas online.

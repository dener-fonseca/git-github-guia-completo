# 🌍 Origem do GitHub

## 📖 Quando e como o GitHub surgiu?

O **GitHub** foi lançado oficialmente em **19 de outubro de 2008** por:

- **Tom Preston-Werner**
- **Chris Wanstrath**
- **PJ Hyett**

O objetivo da plataforma era tornar o uso do Git mais simples, acessível e colaborativo para desenvolvedores do mundo inteiro.

O projeto foi criado por programadores que acreditavam que o Git poderia se tornar muito mais intuitivo através de uma plataforma online moderna e colaborativa.

## 💡 O problema da época

Antes do GitHub, o Git já era utilizado como sistema de controle de versão, porém seu uso era muito mais técnico e limitado ao ambiente local.

<p align="center">
  <img src="https://git-scm.com/images/logos/downloads/Git-Icon-1788C.png" width="140px" alt="Git Logo">
</p>

Faltava uma plataforma que permitisse:

- hospedar repositórios online;
- compartilhar projetos facilmente;
- colaborar entre equipes;
- revisar código de forma visual;
- acompanhar mudanças pela web.

Foi nesse cenário que surgiu o GitHub.

---

## 🚀 A grande ideia do GitHub

O GitHub trouxe uma interface web moderna construída sobre o Git, adicionando recursos sociais e colaborativos que revolucionaram o desenvolvimento de software.

<p align="center">
  <img src="https://github.githubassets.com/images/modules/logos_page/GitHub-Mark.png" width="140px" alt="GitHub Logo">
</p>

Com o GitHub, os desenvolvedores passaram a conseguir:

- publicar projetos na internet;
- colaborar em equipe;
- contribuir para projetos open source;
- revisar código online;
- acompanhar alterações em tempo real.

---

## 🕰️ Linha do tempo

| Ano | Evento |
|------|---------|
| **2008** | Lançamento oficial do GitHub |
| **2010** | Crescimento acelerado da comunidade open source |
| **2018** | Microsoft adquire o GitHub por aproximadamente 7,5 bilhões de dólares |
| **Atualmente** | GitHub se torna uma das maiores plataformas de desenvolvimento do mundo |

---

## 🌐 Impacto do GitHub

O GitHub transformou completamente a forma como projetos de software são desenvolvidos.

A plataforma popularizou conceitos como:

- colaboração online;
- open source;
- Pull Requests;
- revisão de código;
- integração entre equipes globais.

Hoje, milhões de desenvolvedores utilizam o GitHub diariamente para criar, compartilhar e manter projetos de software.

---

## 🧠 Curiosidades

- O GitHub começou como um projeto pequeno entre amigos desenvolvedores.
- A plataforma rapidamente se tornou referência mundial em projetos open source.
- Em 2018, a Microsoft adquiriu o GitHub por aproximadamente **7,5 bilhões de dólares**.
- Atualmente, milhões de repositórios estão hospedados no GitHub.

---

## ✅ Resumo

O GitHub nasceu com o objetivo de tornar o Git mais acessível e colaborativo.

Ao unir controle de versão, hospedagem de código e ferramentas de colaboração em uma única plataforma, o GitHub revolucionou o desenvolvimento de software moderno 🚀

# 🎛️ O que o GitHub faz?

O **GitHub** é uma plataforma de hospedagem de código baseada no Git que permite armazenar projetos na nuvem e colaborar com outros desenvolvedores de forma simples e organizada.

Além do controle de versão, o GitHub adiciona recursos sociais, ferramentas de colaboração e automação para o desenvolvimento de software.

---

# 🔄 Git vs GitHub

Embora estejam relacionados, Git e GitHub não são a mesma coisa.

| Git | GitHub |
|------|---------|
| Ferramenta de controle de versão | Plataforma online baseada em Git |
| Funciona localmente | Funciona na nuvem |
| Gerencia histórico do projeto | Facilita colaboração e compartilhamento |
| Utilizado pelo terminal | Possui interface web |
| Distribuído e gratuito | Plataforma com recursos gratuitos e pagos |

---

# 🌍 Para que serve o GitHub?

O GitHub possui diversas funções importantes no desenvolvimento moderno.

---

## 📦 1. Hospedar projetos

O GitHub permite armazenar repositórios online de forma pública ou privada.

Com isso, você pode:

- acessar projetos de qualquer lugar;
- manter backups seguros;
- compartilhar código facilmente;
- sincronizar alterações entre dispositivos.

---

## 👥 2. Colaboração entre desenvolvedores

A plataforma facilita o trabalho em equipe através de recursos como:

- Pull Requests;
- branches;
- revisão de código;
- comentários;
- Issues;
- Discussions.

Isso permite que várias pessoas trabalhem simultaneamente no mesmo projeto.

---

## 💼 3. Criar um portfólio profissional

Muitos desenvolvedores utilizam o GitHub como vitrine profissional.

Os repositórios publicados podem demonstrar:

- projetos pessoais;
- conhecimentos técnicos;
- experiência prática;
- participação em projetos open source.

Por isso, o GitHub é muito valorizado no mercado de tecnologia.

---

## ⚙️ 4. Automatizar processos

O GitHub também oferece ferramentas de automação, como o GitHub Actions.

Com elas, é possível:

- executar testes automáticos;
- realizar deploy;
- validar código;
- automatizar tarefas repetitivas.

---

# 💡 GitHub na prática

Fluxo básico de funcionamento:

```text
Projeto Local
      ↓
     Git
      ↓
GitHub
      ↓
Colaboração Online
```

O Git controla as versões localmente, enquanto o GitHub permite compartilhar e colaborar pela internet.

---

# 🌟 Por que o GitHub é tão importante?

O GitHub se tornou uma das plataformas mais utilizadas do mundo porque reúne:

- versionamento;
- colaboração;
- hospedagem de código;
- automação;
- documentação;
- gerenciamento de projetos.

Tudo isso em um único ambiente.

---

# ✅ Resumo

O GitHub pode ser entendido como:

```text
Git + Plataforma de colaboração para desenvolvedores
```

Ou seja, ele transforma o controle de versão do Git em uma experiência colaborativa, online e organizada 🚀

# 🎛️ O que o GitHub faz?

GitHub é uma **plataforma de hospedagem de código** que adiciona **colaboração social** ao Git.

## 🔄 Git vs GitHub

| Git | GitHub |
|-----|--------|
| Ferramenta local | Plataforma online |
| Controle de versão | + Colaboração |
| Terminal | Interface web |
| Gratuito | Freemium |

## 🌍 Para que serve?

1. **Hospedar código** publicamente ou privadamente
2. **Colaborar** com outros desenvolvedores
3. **Mostrar portfolio** profissional
4. **Automatizar** deploy e testes

**Resumo**: GitHub = Git + Redes sociais para programadores
